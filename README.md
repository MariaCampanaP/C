## ☕︎ Sobre o Portugol
**C** é uma linguagem de programação de próposito geral, muito utilizada no ensino de lógica e estrutura de dados por sua proximidade com a linguagem de máquina e seu alto desempenho. Embora seja mais técnica que linguagens didáticas como o Portugol, C é essencial para quem deseja se aprofundar na programação, principalmente nas áreas de sistemas, embarcados e cibersegurança. Seus principais contéudos incluem declaração de variáveis (como **int**, **float**, **char**, entre outros), operadores matemáticos, lógicos e relacionais, estruturas condicionais (como **if**, **else if** e **else**, para tomadas de decisão), laços de repetição (como **while**, **for**, e **do while**), funções que permitem reutilização de código e organização de programas, vetores e matrizes usados para armazenar coleções de dados e manipulação de arquivos, ponteiro e alocação dinâmica de memória, que oferecem maior controle sobre os recursos do sistema.
Apesar de exigir mais atenção com detalhes como ponto e vírgula, chaves e gerenciamento de memória, **aprender C desenvolve uma base sólida de programação e raciocínio computacional**. É uma linguagem poderosa e versátil, usada em sistemas operacionais, compiladores e jogos.
